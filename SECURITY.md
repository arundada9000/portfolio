# Security Policy

## Supported Versions

Only the latest deployed version of the portfolio (at [arunneupane.vercel.app](https://arunneupane.vercel.app)) is supported with security updates.

## Reporting a Vulnerability

If you discover a security vulnerability in this portfolio, please report it privately.

**Do not open a public GitHub issue** for security-related concerns.

Reach out directly:

- **Email**: arunneupane0000@gmail.com
- **Subject**: "Security: [brief description]"

I aim to acknowledge receipt within 48 hours and will work on a fix as soon as possible. Thank you for helping keep this project secure.
